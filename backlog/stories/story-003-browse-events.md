# Story 003 — Browse events

**Epic:** [epic-002-events](../epics/epic-002-events.md)  
**Status:** Ready  
**Sprint:** [Sprint 02](../../sprints/sprint-02/sprint.md)

---

## User story

As a **user** (any actor, including unauthenticated spectators),  
I want to browse upcoming events,  
so that I can find tournaments, leagues, camps, and showcases relevant to me.

---

## Acceptance criteria

- [ ] Events list is accessible without signing in
- [ ] Events can be filtered by: date range, location, event type (T / L / K / Sh), age group
- [ ] Each event in the list shows: name, type, date(s), location, and registration status
- [ ] List is paginated or uses infinite scroll
- [ ] Works on mobile and desktop

---

## Notes

- Event types: Tournament (T), League (L), Camp/Clinic (K), Showcase (Sh)
- See [../../domain/event-types.md](../../domain/event-types.md) for event type definitions
