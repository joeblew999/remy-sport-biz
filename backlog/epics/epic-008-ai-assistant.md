# Epic 008 — AI Assistant

**Status:** In backlog  
**Roadmap phase:** AI Assistant  
**Actors:** Organizer, Admin (write) · All (Q&A read)  
**Event types:** All

---

## Goal

Organizers can create and configure events through a chat interface. AI can suggest bracket structures. All users can ask questions about standings, schedules, and results.
